---
description: List unprocessed meeting transcripts
---

Use the `list_unprocessed_transcripts` MCP tool to show all meeting transcripts that haven't been processed yet.

$ARGUMENTS

If the user specified a number of hours, pass that as `older_than_hours`. Otherwise, list all unprocessed transcripts.

Display the results in a clear table showing:
- Date
- Title
- Duration
- Attendees (if any)
